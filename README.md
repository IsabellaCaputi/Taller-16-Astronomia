1. Colocar en el mismo directorio:
   - analisis_estructuras.py
   - galaxy_cartesian_coordinates.ecsv
2. Ejecutar:
   python analisis_estructuras.py
3. Resultados generados automáticamente en el directorio:
   catalogo_clusters.ecsv, catalogo_voids.ecsv, trazado_filamentos.ecsv,
   estadisticas.txt, distribucion_galaxias.pdf, estructuras_clasificadas.pdf,
   clusters_y_voids.pdf, red_cosmica_completa.pdf
4. Parámetros editables al inicio del script: N_VECINOS, DISTANCIA_AGRUPACION_CLUSTER,
   DISTANCIA_AGRUPACION_VOID, DISTANCIA_MAXIMA_FILAMENTO, PASO_MUESTREO_FILAMENTO.
